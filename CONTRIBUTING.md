# Contributing to True Deep Search

Thank you for your interest in contributing.

This project aims to become a real hybrid deep search engine, and contributions are welcome from developers who want to build something meaningful and technically solid.

---

## Project Philosophy

Before contributing, please understand the core principles:

- This is **not a simple CLI tool**
- This is a **search engine system**
- Architecture matters
- Performance matters
- Clean code matters
- UI should stay responsive and minimal

---

## What You Can Contribute

We welcome contributions in the following areas:

### 1. Providers
- Add new search providers (API-based or scraping-based)
- Improve existing providers
- Normalize results better

### 2. Ranking
- Improve scoring logic
- Add query intent detection
- Improve source weighting

### 3. Indexing
- Improve local indexing
- Add new document types
- Improve performance

### 4. UI / UX
- Better layouts
- Improved navigation
- Preview panels
- Filters and tabs

### 5. Infrastructure
- Logging improvements
- Config enhancements
- HTTP optimizations
- Error handling

### 6. Documentation
- Improve README
- Add examples
- Add architecture docs
